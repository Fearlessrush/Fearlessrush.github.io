Linux基本操作

发表于 2019-07-10

cd / 根目录
cd .. 返回上一级
cd ~ 进入home
cd - 返回到上一次的工作目录
pwd 目录路径
ls 查看目录有哪些文件夹
ls -a 显示所有文件包括隐藏文件
ls -l 显示详细列表
ls -lh 把文件大小显示出来
ll = ls -l 查询当前目录下所有文件夹

mkdir 文件夹名 创建这个文件夹
mkdir /home/chuang/shipin 创建在home下的chang下创建shipin 如果没有父级目录 -p
mkdir a b 同时创建 a b两个文件夹
mkdir a/{c,d} 进入a目录下 同时创建 c d两个文件夹
mkdir .abc 创建.abc 隐藏文件

. 代表当前目录
.. 代表上一级目录

注意要进入home 目录创建文件夹 其他目录不要随意创建
普通用户在根目录创建文件夹没有权限

Touch 创建
touch 文件名 创建文件 也可以指定目录 但必须保证父级目录存在

rm 删除
rm 加文件名 删除这个文件
rm a b 删除多个文件
rm abc -r 删除这个文件夹
rm * -r 删除当前文件夹所有内容 不能删除隐藏文件

gedit直接用记事本方式打开
gedit a.txt 打开a.txt 记事本

clear 清屏

命令敲错了 按ctrl + c 结束命令
查看帮助命令 ls –help

Rwx (权限)
d rwx rwx r-x d代表文件夹 -代表文件
rwx rwx r-x r可读 w 可写 x 可执行
分为三组 第一组代表文件拥有者的权限 第二组代表文件拥有组的权限 第三组代表其他用户的权限
tmp 是公共的 任何人在里面都有所有权限

cp(查找)

比如创建 a ab abc
ls a* 查找以a打头的文件
ls a? ?a代表任意一个字符
ls a?? ??代表两个任意字符

cp(拷贝)

cp 拷贝文件名 拷贝到那个目录
cp 1.txt nan
cp 1.txt 2.txt -a 拷贝1.txt所有信息 改名为2.txt
拷贝文件目录
cp nan nv -r

mv(移动)
mv 1.txt nv 移动1.txt 到nv文件夹里
mv 2.txt nv/3.txt 移动2.txt 到nv文件夹中改名为3.txt

覆盖 >>追加(重定向)
Ls查询内容写入 > 1.txt信息覆盖
Pwd 查询内容写入 >>1.txt 信息追加到末尾

Cat(查看)
Cat a.txt 查看内容
Cat a.txt b.txt 查询多个文件名
Cta a.txt b.txt > c.txt 把a.txt和b.txt信息合并把结果写入c.txt

More (分页)
More 文件名 按空格 或者回车键下一页 返回上一页 按B
结束按 q退出
管道 |
举例 ls –al / |more 把左边返回的命令返回的数据 交给右边的more处理

Ln 链接 类似windows快捷方式
软链接：只记录源文件的名称 （路径）不记录源文件内容
软连接 ln –s
ln –s文件名或者文件夹 快捷方式名称
给其他目录创建软链接 ，如果源文件不写绝对路径 那么创建的方式会无效
硬链接 ln
ln文件名（不能是目录）链接名称 相当于拷贝 删除源文件也不会影响硬链接

Histor 查看输入命令的历史
查找文件内容 (grep)
grep文本搜索
grep 搜索内容 搜索那个文件
grep –i 不区分大小写
grep –v 反向查找
grep –n hello . –r 查询当前目录所有文件中含有hello的内容
以^a a开头
以o$ o结尾

查找文件（find）
find /home –name 1.txt 查找home文件夹下 名称为1.txt的文件
find /home –name ‘*txt’ 查找home文件夹下 后缀名称为txt的文件

归档管理：压缩文件 （tar）
打包文件：tar cf 打包文件名称.tar 要打包的内容
解包文件：xvf txt.tar 如果没有指定目录解包到当前目录
xvf txt.tar –C tar 指定文件夹
查看打包文件的内容：tar tf txt.tar
查看打包文件的内容进度：tar cvf txt.tar 1.txt 2.txt

压缩、解压文件（gzip）

1.打包
tar –cf txt.tar 1.txt 2.txt 生成txt.tar
2.压缩
gzip –r txt.tar 生成txt.tar.gz
3.解压
gzip –d txt.tar.gz 生成txt.tar
4.解包
tar xvf txt.tar –C test 解包到指定目录

bzip2 压缩
tar jcvf test.tar.bz2 aa bb cc 打包压缩aa bb cc
bzip2 解压
tar xjvf test.tar.bz2 –C test 大写C 指定解压到那个目录

zip 压缩
zip –r zz（压缩名称 不需要写扩展名） aa bb cc 压缩aa bb cc

unzip  -d  zz(目录 如果没有自动创建)  zz.zip（压缩文件）

su 切换管理员

Vim基本操作
命令

简单说明

:w

保存编辑后的文件内容，但不退出vim编辑器。这个命令的作用是把内存缓冲区中的数据写到启动vim时指定的文件中。

:w!

强制写文件，即强制覆盖原有文件。如果原有文件的访问权限不允许写入文件，例如，原有的文件为只读文件，则可使用这个命令强制写入。但是，这种命令用法仅当用户是文件的属主时才适用，而超级用户则不受此限制。

:wq

保存文件内容后退出vim编辑器。这个命令的作用是把内存缓冲区中的数据写到启动vim时指定的文件中，然后退出vim编辑器。另外一种替代的方法是用ZZ命令。

:wq!

强制保存文件内容后退出vim编辑器。这个命令的作用是把内存缓冲区中的数据强制写到启动vim时指定的文件中，然后退出vim编辑器。

ZZ

使用ZZ命令时，如果文件已经做过编辑处理，则把内存缓冲区中的数据写到启动vim时指定的文件中，然后退出vim编辑器。否则只是退出vim而已。注意，ZZ命令前面无需加冒号“：”，也无需按Enter键。

:q

在未做任何编辑处理而准备退出vim时，可以使用此命令。如果已做过编辑处理，则vim不允许用户使用“:q”命令退出，同时还会输出下列警告信息：

No write since last change (:quit! overrides)

:q!

强制退出vim编辑器，放弃编辑处理的结果。如果确实不需要保存修改后的文件内容，可输入“:q!”命令，强行退出vim编辑器。

:w filename

把编辑处理后的结果写到指定的文件中保存

:w! filename

把编辑处理后的结果强制保存到指定的文件中，如果文件已经存在，则覆盖现有的文件。

:wq! filename

把编辑处理后的结果强制保存到指定的文件中，如果文件已经存在，则覆盖现有文件，并退出vim编辑器。
