LAMP环境安装和根据端口配置多个项目

发表于 2019-07-13

1 安装wget
如果系统已安装wget,请跳过此步骤
yum install wget

2 更新系统

#更新yum软件包
yum check-update

#更新系统
yum update

3禁用防火墙
systemctl status firewalld.service ##查看防火墙
systemctl stop firewalld.service ##停止防火墙
systemctl disable firewalld.service ##开机自启服务 禁用

4安装acache
4-1 安装Apache
yum install httpd
4-2 启动Apache
systemctl start httpd.service
4-3 设置Apache开机启动
systemctl enable httpd.service
4-4 查看Apache版本信息
Httpd-v

5安装mysql

5-1 下载mysql5.7的rpm文件

wget http://dev.mysql.com/get/mysql57-community-release-el7-8.noarch.rpm
5-2 安装rmp文件
yum localinstall mysql57-community-release-el7-8.noarch.rpm
5-3 安装MySQL数据库
yum install mysql-community-server
5-4 启动数据库
systemctl start mysqld
6-5 查看数据库密码
grep ‘temporary password’ /var/log/mysqld.log
5-6 进入数据库
mysql -u root -p //回车输入密码 输入不显示密码
5-7 修改数据库初始密码
set password for ‘root’@’localhost’=password(‘Km@3814143’);
5-8 修改数据库远程连接权限
use mysql; //使用mysql 这个库
grant all privileges on wedo.* to ‘root(用户名)‘@’%’ identified by ‘密码’ with grant

GRANT ALL PRIVILEGES ON . TO ‘root ‘@’%’ IDENTIFIED BY ‘Km@3814143’ WITH GRANT OPTION;

option; //开启所有IP连接访问权限
flush privileges;//刷新权限
exit //退出数据库
5-9 //重启数据库
service mysqld restart

6安装PHP7
1安装依赖包
yum install gcc-c++ libxml2 libxml2-devel openssl openssl-devel bzip2 bzip2-devel libcurl libcurl-devel libjpeg libjpeg-devel libpng libpng-devel freetype freetype-devel gmp gmp-devel libmcrypt libmcrypt-devel readline readline-devel libxslt libxslt-devel
2安装php组新版本rpm
rpm -Uvh https://mirror.webtatic.com/yum/el7/epel-release.rpm
rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm
3查看有哪些php最新版的安装包文件
yum search php71
4安装php7最新版
yum install mod_php71w php71w-mysqlnd php71w-cli php71w-fpm
5查看php 版本信息
php -v
6重启Apache 服务
systemctl restart httpd
7测试环境
vim /var/www/html/index.php
编辑

网站根目录在 var/www/html
Apache配置目录在 etc/httpd/conf/httpd.conf
配置端口和访问路径
Listen 81
Listen 82

<VirtualHost *:81>

DocumentRoot /var/www/html/wordpress

<VirtualHost *:82>

DocumentRoot /var/www/html/edusoho/web
重启apache service httpd restart
