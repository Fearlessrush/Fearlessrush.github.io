sql基本语句

发表于 2019-07-13

SQL SELECT 语法
SELECT 列名称 FROM 表名称
以及：
SELECT * FROM 表名称
注释：SQL 语句对大小写不敏感。SELECT 等效于 select。

Where条件

操作符 描述
= 等于
！= 不等于

大于
< 小于
= 大于等于
<= 小于等于
BETWEEN （between） 在某个范围内
LIKE (like) 搜索某种模式

SELECT DISTINCT 列名称 FROM 表名称
屏蔽重复的数据 只看不同的 不看相同的数据
distinct
Select distinct user from users
展示user这一行 不重复的数据 来自users这个表

SELECT * from users where year >1995
查询这个表中 年份大于1995年的全列数据
SELECT *from users where user=’3’
查询这个表中 用户是3的全列数据

And
SELECT * FROM test WHERE name=’小花’ and age =’34’
查询test这个表里叫小花 并且24岁的数据
Or
SELECT * from test WHERE name =’小米’ or age=’12’
查询test 这个表里数据包含小米 或者年龄 在12岁的数据
Or 和 and 组合
SELECT * from test WHERE(age=’12’ or age=’34’)and name=’小米’
查询test 这个表里 年龄是12 或者 34 同时都叫小米的数据

Between 之间

Select * from test where age between age 17 and 18
查询 test 这个表里 年龄 在17 到 18 之间的全部内容

SELECT 产品名称,单价 from 产品 ORDER BY 单价

ORDER BY 语句
排序 order by
升序 ase
降序 desc

Select name,age from test order by age
根据 名字 和年龄 查找 test表 对 年龄从小到大排序
SELECT name,age from test ORDER by age DESC
根据 名字 和年龄 查找 test表 对 年龄从大到小排序 DESC逆顺序
SELECT name,age from test ORDER by age DESC,name ASC
根据 名字 和年龄 查找 test表 对 年龄从大到小排序 DESC逆顺序 ASC字母顺序
SQL INSERT INTO 语句

INSERT INTO 语句
INSERT INTO 语句用于向表格中插入新的行。
语法
INSERT INTO 表名称 VALUES (值1, 值2,….)
我们也可以指定所要插入数据的列：
INSERT INTO table_name (列1, 列2,…) VALUES (值1, 值2,….)

INSERT INTO test VALUES(‘6’,’小猫’,’123’,’34’,’女’)
插入 test (‘6’,’小猫’,’123’,’34’,’女’)
INSERT INTO test(ID,name,pwd,age,sex) VALUES(‘7’,’小王’,’123’,’34’,’女’)
插入 test 对应字段 对照相应的数据 字段用逗号， 数值用单引号

Update 语句
Update 语句用于修改表中的数据。
语法：
UPDATE 表名称 SET 列名称 = 新值 WHERE 列名称 = 某值
UPDATE test set name=32423 WHERE ID =7
修改test 中 名字 第七行

UPDATE test set name=”小黑”,pwd=”没有密码” where id=7
修改 test表中 name 和密码字段的值 根据id修改

UPDATE test set pwd=”6666” WHERE name=”小草”
修改 test 表中 pwd 的值 根据 name修改

DELETE 语句
DELETE 语句用于删除表中的行。
语法
DELETE FROM 表名称 WHERE 列名称 = 值

DELETE FROM test WHERE id=1
删除id 为1的一行

删除所有行
可以在不删除表的情况下删除所有的行。这意味着表的结构、属性和索引都是完整的：
DELETE FROM 表名（test）

Like 模糊查询
SELECT * from test WHERE name like ‘小%’
查询 表里 有小的 内容
SELECT * from test WHERE name like ‘小_’
查询 表里 小字开头后面一个字符的内容
SELECT * from test WHERE name like ‘_小’
查询 表里 小字结尾前面一个字符的内容

聚合函数
Count() 查询几个id
Select count(id) from tets
查询id 有多少个
Max() 查询年龄里最大的
Select max(age) from tets
Min()查询年龄里最小的
Select min(age) from test
Sum()查询年龄的总和
Selecr sum(age) from test
Avg()查询年龄的平均值
Select avg(age) from tets

分组：
Group by

分页 limit n,m
Select * from test limit 0,3
查询下标从0开始到3的数据

约束
Primary key 主键 唯一不能为空
Unqque 唯一
Auto _increment 自增
Not_null 不为空
CREATE table test2(id int PRIMARY key auto_increment,name VARCHAR(10),age int(3))
创建 一个表为test2 id整型 主键唯一不能为空 自增

两表联查
99查询
select test2.name,test3.name FROM test2,test3 where test2.id=test3.id
select * from test2,test3 WHERE test2.id= test3.id
内敛查询
select * from test2 join test3 on test2.id = test3.id
三表联查
select * from test,test2,test3 WHERE test.id = test2.id = test3.id

select * from test JOIN test2 JOIN test3 on test.id = test2.id = test3.id

左外链接和右外链接：
select * from test2 left JOIN test3 on test2.id = test3.id
